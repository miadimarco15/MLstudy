# Kaggle Projects

Self-directed machine learning projects built on public Kaggle datasets. Each
project lives in its own folder with a dedicated README describing the data,
approach, and results.

All datasets used here are public, so they can be downloaded directly from the
links in each project README. The notebooks are provided with outputs and figures
pre-rendered, so the analyses are viewable without running them.

## Projects

| Project | Topic | Stack | Type |
|---|---|---|---|
| [stellar-classification](./stellar-classification) | Classifying astronomical objects (galaxy / quasar / star) from photometric data | Python, XGBoost | Multi-class classification |
| [titanic-survival](./titanic-survival) | Predicting passenger survival on the Titanic | Python, scikit-learn | Binary classification |

## Tools

- **Language:** Python
- **Libraries:** NumPy, pandas, scikit-learn, XGBoost, matplotlib, seaborn
